# Arabic BiDi Fixer

مشروع مفتوح المصدر لحل مشكلة تشوه النص العربي في منصات الدردشة.

## المكونات
- **arabic-bidi-fixer-auto**: إضافة تلقائية.
- **arabic-bidi-fixer-manual**: إضافة يدوية.
- **bidi-test-suite.html**: صفحة اختبار شاملة.

## التثبيت
1. حمل المجلد المطلوب.
2. اذهب إلى chrome://extensions.
3. فعّل Developer mode.
4. اضغط Load unpacked واختر المجلد.
