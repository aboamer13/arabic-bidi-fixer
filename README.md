# Arabic BiDi Fixer

![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)
![Version](https://img.shields.io/badge/version-1.1-blue)

حل مفتوح المصدر لمشكلة تشوه النص العربي في منصات الدردشة التي تعتمد على الذكاء الاصطناعي (مثل ChatGPT، DeepSeek، Claude). يقوم بإصلاح اتجاه النص والمؤشر ويعالج ظاهرة "العرض ≠ النسخ" باستخدام إدراج حرف `\u200F` ديناميكيًا.

## 📦 المكونات
- **arabic-bidi-fixer-auto**: إضافة تلقائية تعمل على جميع المواقع، آمنة ولا تسبب تجميدًا (تستخدم `requestAnimationFrame`). تدعم Shadow DOM وصفحة إعدادات للقائمة السوداء.
- **arabic-bidi-fixer-manual**: إضافة يدوية تعمل بكبسة زر، مثالية لمن يريد تحكمًا كاملاً.
- **bidi-test-suite.html**: صفحة اختبار شاملة تحتوي على حالات BiDi معقدة.
- **test.sh**: سكريبت لفتح صفحة الاختبار تلقائيًا.
- **RESEARCH_PAPER.md**: ورقة بحثية توثق الاكتشاف والحل.

## 🔧 التثبيت
1. حمّل المجلد الذي تريده (Auto أو Manual) من المستودع.
2. افتح Chrome واذهب إلى `chrome://extensions`.
3. فعّل "Developer mode" (وضع المطور).
4. اضغط "Load unpacked" (تحميل غير مضغوط) واختر المجلد.
5. الإضافة جاهزة للعمل فورًا.

## ✨ كيف يعمل؟
- تطبيق `direction: rtl` و `unicode-bidi: plaintext` على العناصر التي تحتوي على نص عربي.
- إدراج حرف `\u200F` (Right-to-Left Mark) حول المقاطع العربية لإجبار طبقة العرض على الترتيب الصحيح دون التأثير على النص الأصلي.
- مراقبة DOM باستخدام `MutationObserver` مع تجميع التغييرات عبر `requestAnimationFrame` لتجنب أي تجميد.
- دعم Shadow DOM (في الإصدار التلقائي).
- صفحة إعدادات تسمح بتعطيل الإضافة على مواقع محددة (في الإصدار التلقائي).

## 🧪 الاختبار
افتح `bidi-test-suite.html` في المتصفح، أو شغّل `test.sh`. تأكد من أن جميع النصوص تظهر بشكل سليم، وجرب النسخ واللصق.
